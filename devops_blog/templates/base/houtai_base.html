开发步骤一: 创建python virtualenv 这里不再赘述

环境

操作系统:linux

Python版本:python3.7

flask版本:1.0.2 --- 快速上手中的例子如下:

	from flask import Flask
	app = Flask(__name__)
	
	@app.route('/')
	def hello_world():
		return 'Hello, World!

项目创建好了

下面运行flask的代码了


		$ export FLASK_APP=hello.py
		$ flask run
	 	* Running on http://127.0.0.1:5000/
